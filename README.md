## Installations
* nodejs
* npm
* angularCLI
* typescript

## Setup
Run ``https://github.com/Atienodolphine/Git-Explore.git``

or download the zip file from github.

After extracting the files, run 

* 1  .`` cd Gitsearch.`` 
* 2  .``ng serve.``

you can also run tests with ``ng test.``
